# 创建复杂报表

## 创建复杂报表⼊⼝

平台提供多种便捷⼊⼝快速进⼊复杂报表创建流程， ⽀持创建专业级SmartBI 数据分析报表， ⽤⼾可根据分析需求的复杂程度选择合适的⼊⼝和报表类型。

**⽅式⼀： ⾸⻚快速创建⼊⼝**

1. 登录智能数字员⼯平台后， 默认进⼊⾸⻚

2. 在⾸⻚中间区域找到"快速创建⼊⼝"模块

3. 点击【复杂报表】 快捷按钮
4. 系统弹出应⽤选择对话框， 选择需要挂载报表的⽬标应⽤
5. 确认后跳转到复杂报表创建⻚⾯，进⼊数据源配置步骤

**适⽤场景：** ⽤⼾需要创建专业级多维数据分析报表。

![image-20260123101802881](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123101802881.png)

**⽅式⼆： 应⽤⻚⾯创建**

1. 点击导航栏的【我的应⽤】 进⼊应⽤管理⻚⾯

2. 选择需要添加复杂报表的应⽤， 点击应⽤名称进⼊应⽤开发⻚⾯
3. 在低代码设计器中点击【新建⻚⾯】按钮
4. 在创建⽅式选择菜单中， 选择【复杂报表】
5. 直接进⼊数据源配置和SmartBI报表设计流程

**适⽤场景：** 为现有应⽤添加专业数据分析能⼒。

![image-20260123101827835](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123101827835.png)

 **方式三：通过AI智能助⼿创建**

1. 在平台任意⻚⾯， 点击右下⻆的AI智能对话助⼿图标
2. 在对话框中描述复杂分析需求， 如"我需要创建多维透视分析报表"
3. AI助⼿理解需求后， 会推荐合适的 SmartBI报表类型和⽬标应⽤
4. 确认信息后， AI助⼿会引导进⼊复杂报表创建流程

**适⽤场景：** ⽤⼾需要AI协助选择合适的报表类型。

## 数据集管理

⽀持连接外部数据源或快速选择应⽤已连接或历史应用下本人创建的数据源。 平台会⾃动在SmartBI中创建应⽤⽂件夹， 并将数据源归结⾄该⽂件夹下。

![image-20260123102111890](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102111890.png)

**步骤一：添加数据源**

在报表创建向导的第一步，需配置分析所需的数据源。

- **新建外部数据源：** 点击"新建数据源"，选择数据库类型（PolarDB、MySQL、Oracle、PostgreSQL等），填写连接信息包括主机IP地址、端口号、数据库名称、用户名和密码。点击"测试连接"确认连接成功后保存，然后可选择需要分析的数据表。

  ![image-20260123102215002](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102215002.png)

- **选择应用已连接数据源：** 点击"选择已连接数据源"，可直接使用应用连接过的数据源，也可直接选择到历史应用下本人创建的数据源，快速开始报表制作。注：可选择本应用连接及本人历史连接的数据源。

![image-20260123102243418](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102243418.png)

- **管理操作：** 支持对数据源进行重命名、连接测试、刷新表结构及删除操作。

**步骤二：选择数据表并推送至SmartBI**

在数据源列表中展开数据源，浏览可用的数据表，勾选需要进行分析的一个或多个数据表，点击"下一步"按钮，将选择的数据表同步到SmartBI平台进行后续分析。

![image-20260123102430004](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102430004.png)

## 选择SmartBI报表类型

根据分析需求选择合适的SmartBI报表类型：

- **即席查询：** 适用于临时性、探索性的数据查询分析，支持拖拽字段快速生成统计报表
- **交互式仪表盘：** 适用于综合性可视化看板，支持多图表组合展示和交互式筛选分析
- **透视分析：** 适用于多维度交叉分析，支持数据钻取、切片、旋转等OLAP操作
- **Web电子表格：** 适用于类似Excel格式的复杂报表制作，支持公式计算和精细化格式控制

页面默认打开即席查询的SmartBI报表编辑页面，编辑去可通过用户的选择自动切换到相应的报表类型编辑区。

![image-20260123102537252](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102537252.png)

## SmartBI报表编辑

在SmartBI编辑器中完成专业级报表设计和数据建模。

![image-20260123102659919](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102659919.png)

**创建数据模型：**

1. 在SmartBI编辑器中打开数据模型设计界面
2. 点击右侧栏中“选择业务数据集”时，可从数据源中选择到该应用推送的数据集相关数据表进行分析，或点击弹窗上点击“创建数据模型”按钮进入数据模型创建页面
3. 通过拖拽连线建立表与表之间的关联关系
4. 配置字段属性、设置主键和外键关系
5. 保存数据模型以供后续分析使用

![image-20260123102741552](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102741552.png)

**创建分析图表：**

1. 根据选择的报表类型进入相应的设计界面
2. 从字段列表拖拽维度和度量到图表设计区域
3. 选择合适的图表类型（柱状图、折线图、饼图等）
4. 配置图表样式、颜色主题和交互功能
5. 预览图表效果并进行调整优化
6. 保存分析图表：点击保存可见在SmartBI的分析报表目录下有平台自动生成的相应的应用名称的文件下，可在保存时将分析图表保存至该文件下，也可手动创建所需的报表

![image-20260123102816260](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123102816260.png)

<div style="border: 1px solid #FFC402; background-color: #FFFBED; padding: 12px; border-radius: 4px; margin-top: 10px;"> <strong>⚠️注意事项：</strong><br>报表编辑完成后必须点击SmartBI的保存按钮才能⽣效<br></div>

**挂载创建的数据源至应用下：**

可在保存在文件下选择到保存的分析图表，勾选图表点击“保存并完成”按钮后可返回至应用编辑页，可直接在该页面进行后续分析图表的调整和修改；**注：勾选图表时只可勾选一个图表，目前无法勾选多个图表。**

![image-20260123103129357](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123103129357.png)

**编辑器操作：**

- **保存：** 点击"保存"按钮，将报表设计内容持久化存储。**注意：必须在SmartBI内保存才能生效。**
- **预览：** 点击"预览"，查看使用端的报表展示效果
- **访问应用：**点击"访问应用"，可直接打开应用的测试环境进行访问测试
- **刷新：**点击"刷新"按钮，同步数据源的最新表结构变更
- **关闭：**点击"关闭"按钮，可关闭当前报表页面，且回到应用下低代码设计器页面

# 修改复杂报表

当需要对已创建的复杂报表进行调整和优化时，平台提供了完善的修改功能，支持对报表的各个环节进行灵活调整。

## **修改复杂报表入口：**

* 点击导航栏的【我的应用】进入应用管理页面
* 选择目标应用，进入应用开发页面
* 在应用的报表列表中找到需要修改的复杂报表
* 点击报表名称直接进入SmartBI编辑器进行修改

## 数据源调整

对已创建报表的数据源配置进行修改，以适应业务数据变化需求。

### **添加新数据源：**

1. 进入报表编辑页面，点击【SmartBI数据源】选项卡
2. 点击"添加数据源"按钮，可填写新的数据源连接信息并连接，也可选择应用下其他数据源或选择到历史应用下本人创建的数据源
3. 测试连接并保存成功后，选择需要分析的数据表
4. 点击"推送至SmartBI"，将新数据表同步到分析平台

![image-20260123103635160](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123103635160.png)

### **修改数据表推送：**

1. 展开已配置的数据源，查看当前推送的数据表
2. 勾选数据表，调整推送范围
3. 点击"推送至SmartBI"按钮，更新SmartBI中的数据表结构

**注：**已被推送的数据源表不可进行勾选，且显示已同步状态

![image-20260123103709908](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123103709908.png)

## 数据模型修改

调整表之间的关联关系和字段属性，优化数据分析逻辑。

**调整表关联关系：**

1. 在应用编辑页选中所需调整的SmartBI报表，可直接在智能平台内打开报表编辑页
2. 在SmartBI编辑器中打开数据模型设计界面
3. 选择需要修改的表关联连线，双击打开关联配置
4. 调整关联字段、关联类型（内连接/左连接/右连接）
5. 预览关联结果，确认数据正确性后保存修改

![image-20260123103922972](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123103922972.png)

**修改字段属性：**

1. 在数据模型中选择需要调整的字段
2. 在属性面板中修改字段类型（维度/度量/计算字段）
3. 设置字段的数据格式、聚合方式等属性
4. 保存字段属性修改，系统自动更新相关图表

![image-20260123103939738](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123103939738.png)

## 保存

完成报表修改后，进行保存操作，确保修改内容生效。

**保存修改：**

1. 在SmartBI编辑器中点击【保存】按钮
2. 系统提示保存成功，修改内容已持久化存储
3. 返回应用前端页面，可看到报表最新样式

![image-20260123104016313](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260123104016313.png)
