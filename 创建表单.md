# 创建表单

## 创建表单入口

平台提供多种便捷入口快速进入表单创建流程，支持AI创建表单和上传已有的Excel创建表单两种方式，用户可根据实际需求选择最适合的创建方式和入口。

**方式一：首页快速创建入口**

1. 登录智能数字员工平台后，默认进入首页
2. 在首页中间区域找到"快速创建入口"模块
3. 根据需求选择点击【Excel创建表单】或【AI创建表单】快捷按钮
4. 平台弹出应用选择对话框，选择需要挂载表单的目标应用
5. 确认后跳转到相应的表单创建页面

**适用场景**：用户刚登录系统，希望快速开始创建表单。

![image-20260122181116895](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181116895.png)

**方式二：应用页面创建**

1. 点击导航栏的【我的应用】进入应用管理页面
2. 选择需要添加表单的应用，点击应用名称进入应用开发页面
3. 在低代码设计器中点击【新建页面】按钮
4. 在创建方式选择菜单中，选择【Excel创建表单】或【AI创建表单】
5. 直接进入表单创建流程

**适用场景**：在现有应用中添加新的表单页面。

![image-20260122181158507](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181158507.png)

**方式三：通过AI智能助手创建**

1. 在平台任意页面，点击右下角的AI智能对话助手图标
2. 在对话框中描述表单创建需求，如"我想创建一个员工信息管理表单"
3. AI助手理解需求后，会询问创建方式(AI创建或Excel创建)和目标应用
4. 确认信息后，AI助手会引导进入相应的创建流程

**适用场景**：用户习惯对话式交互，或需要AI协助确定创建方式。

![image-20260122181308732](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181308732.png)

## 功能概述

本功能通过AI对话式交互，实现应用表单页面的快捷创建。平台支持“AI创建表单”和“Excel创建表单”两种模式。

**适用场景：** 业务表单系统搭建（如请假、报销）、数据采集管理、Excel台账数字化改造等。

## AI创建表单

![image-20260122181459214](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181459214.png)

### 操作步骤

1. **输入需求：** 在界面输入框中描述需求，例如：“做一个部门人员的简历管理表单，包含姓名、年龄、学历、简历附件”。
2. **AI优化（可选）：** 点击“AI优化”按钮，AI会对您的简单描述进行丰富和完善，补充可能的字段。
3. **生成设计：** 点击“生成表单设计”按钮，触发AI分析流程。
4. **需求确认：** AI“需求分析助手”会通过结构化提问（如使用对象、流程管理等）进一步梳理细节。

## Excel创建表单

通过上传Excel文件，AI自动解析表头和数据类型，生成表单。

![image-20260122181520772](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181520772.png)

### 操作步骤

1. **文件准备：** 准备好包含数据的Excel文件（.xlsx或.xls格式），确保第一行为表头。
2. **上传文件：** 在上传区域点击选择文件或拖拽文件。
3. **确认上传：** 点击“确认上传”按钮，平台启动AI分析流程。无需额外人工干预。

## 功能设计文档生成

无论哪种创建方式，AI分析后都会生成一份“功能设计文档”。

![image-20260122181817144](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181817144.png)

### 文档内容

包含表单字段（名称、类型、组件类型）、业务规则、流程设计、权限配置及数据字典等。

### 操作选项

- **修改设计：** 支持在页面上直接增删改字段、调整流程。
- **导出文档：** 点击“导出文档”可下载Markdown格式的设计文档。
- **生成功能：** 确认设计无误后，点击“生成功能”或“创建表单”进入下一步。

##  默认数据源

为简化配置流程并确保数据的一致性，平台采用**“默认数据源”**机制。当用户在应用内创建表单时，**平台会自动将表单的数据模型创建在“创建应用时选择或创建的数据源”上**，无需用户手动选择数据源。

### 操作逻辑

- **选择数据源：** 勾选本人创建的数据源。
- **默认创建：** 若不选择直接点击确认，平台会弹出二次确认窗，提示将创建在应用默认数据源中。

## 表单字段调整页面

表单生成后，自动跳转至低代码表单设计页，进行最终的调整和发布。

![image-20260122181934096](https://raw.githubusercontent.com/pinganki37-arch/Intelligent-Digital-Employee-Platform/main/assets/image-20260122181934096.png)

### 核心能力

- **手动修改：** 依附低代码能力，拖拽组件、修改属性、调整布局。
- **AI对话修改：** 在右侧对话框输入指令（如“把姓名改为必填”），AI自动执行修改。
- **批量设置：** 点击“批量设置”，可一键设置全部必填、全部只读或统一字段宽度。
- **预览：** 点击“预览”，查看使用端的列表页和详情页效果（首次预览需等待平台自动发布）。
- **访问应用：**点击“访问应用”，可直接打开应用的测试环境进行访问测试。
- **保存：** 点击“保存”按钮，将修改内容持久化。**注意：除首次生成外，后续修改都需手动保存。**
- **刷新：**点击“刷新”按钮，获取表单最新修改的内容。**注意：未保存的修改会被清空掉。**
- **关闭：**点击“关闭”按钮，可关闭当前表单页面，且回到应用下低代码设计器页面。
